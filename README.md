# MedCap-app
This is an capitation rate app that helps calculate and manage rates, with dashboards, rate-building tools, and VBP performance tracking. This project presents a technology‑driven approach to improving transparency and accuracy in Medicaid Managed Care rate‑setting. It highlights how states determine capitation payments for Managed Care Organizations (MCOs) and identifies key issues in the current system, including fragmented data, complex actuarial models, and limited visibility for providers and patients.

**The Core Problem**:

- States risk overpaying for inefficient care.
  
- MCOs lack clear incentives to improve quality.

- Providers struggle to understand payment structures.

- Patients may receive lower‑quality or poorly coordinated care.

- Rate‑setting relies on complex, inconsistent, and siloed data sources.

**How Rate‑Setting Works**:

- States use financial and encounter data from MCOs.

- Payments are issued as capitation rates—a fixed monthly amount per member.

- Rates must reflect true healthcare costs while encouraging efficient, high‑quality care delivery.

**Why Improvement Is Urgent**:

- Medicaid populations have different cost profiles, requiring nuanced modeling.

- Current systems lack transparency, making it difficult for stakeholders to understand or validate rates.
